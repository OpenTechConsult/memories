# memories

We are going to create a  fully responsive **Full-Stack social Media application** with complete email and Google authentication call **Memories**.
The application will highlight features such as :

- Pagination
- Search and filtering capabilities
- Detail pages 
- Authentication 
- CRUD operations
- Comments
- Etc.

Memories application is an album of memories where people from all around the world can add all the noticeable places they visited or the wildest things they've done. Let's review all the features of the application.

One of the main features is pagination which allows us to fetch a fixed number of memories at a time

- The complete search and filtering feature. We search titles like `Europe`  or search a part of the title like `the` and get all the matching memories/
- The application will also include client-side routing in the memory details page. We can click on memory to get more information about it. We'll event implement the recommended memories section.
